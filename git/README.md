# Git

## get help
```
git help  # in general
git help tutorial  # the very basics

# for some specific command
git help blame
man git-blame
```

## best practice
    git show commit:filepath > location                      # get a file from history to another location
    git for-each-ref --format "%(authorname)" | sort -u      # get all authors
